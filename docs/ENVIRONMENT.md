# Environment Variables Reference

## Backend (`ecommerce-api/.env`)

```env
# ── Application ───────────────────────────────────────────
APP_NAME=Laravel
APP_ENV=local             # local | production | testing
APP_KEY=                  # Generated by `php artisan key:generate`
APP_DEBUG=true            # true for development
APP_URL=http://localhost   # Backend API URL (port 8000)
FRONTEND_URL=http://localhost:3000  # Frontend URL (for PayPal callbacks)

# ── Locale ────────────────────────────────────────────
APP_LOCALE=en
APP_FALLBACK_LOCALE=en
APP_FAKER_LOCALE=en_US

# ── Database ──────────────────────────────────────────
DB_CONNECTION=mysql       # mysql | mariadb | sqlite | pgsql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ecommerce_db
DB_USERNAME=root
DB_PASSWORD=

# ── Session ───────────────────────────────────────────
SESSION_DRIVER=database   # database | file | redis | cookie
SESSION_LIFETIME=120      # Minutes
SESSION_ENCRYPT=false

# ── Queue ─────────────────────────────────────────────
QUEUE_CONNECTION=database # database | redis | sync

# ── Cache ────────────────────────────────────────────
CACHE_STORE=database      # database | redis | file

# ── Mail ────────────────────────────────────────────
MAIL_MAILER=log           # log | smtp | sendmail | mailgun
MAIL_HOST=127.0.0.1
MAIL_PORT=2525
MAIL_USERNAME=null
MAIL_PASSWORD=null
MAIL_FROM_ADDRESS="hello@example.com"
MAIL_FROM_NAME="${APP_NAME}"

# ── Filesystem ─────────────────────────────────────
FILESYSTEM_DISK=local     # local | s3 | public

# ── Redis (optional) ──────────────────────────────────
REDIS_HOST=127.0.0.1
REDIS_PASSWORD=null
REDIS_PORT=6379

# ── AWS S3 (optional) ──────────────────────────────────
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_DEFAULT_REGION=us-east-1
AWS_BUCKET=
```

### Database Configuration

The application supports **MySQL/MariaDB** (production) and **SQLite** (development). The `.env.example` defaults to SQLite.

To use MySQL:
```env
DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=ecommerce_db
DB_USERNAME=root
DB_PASSWORD=your_password
```

### Queue Configuration

The queue driver is set to `database` by default, which uses a `jobs` table. For development, `sync` can be used to process jobs synchronously.

Run the queue worker:
```bash
php artisan queue:listen --tries=1 --timeout=0
```

### Mail Configuration

For development, `MAIL_MAILER=log` writes emails to `storage/logs/laravel.log`.

For production SMTP:
```env
MAIL_MAILER=smtp
MAIL_HOST=smtp.sendgrid.net
MAIL_PORT=587
MAIL_USERNAME=apikey
MAIL_PASSWORD=SG.xxxxx
MAIL_ENCRYPTION=tls
```

## Frontend (`ecommerce-app/.env.local`)

```env
# ── API Configuration ───────────────────────────────────
NEXT_PUBLIC_API_URL=http://localhost:8000/api  # Laravel API URL
NEXT_PUBLIC_STORAGE_URL=http://localhost:8000/storage  # File storage URL
```
